# 자바의 특징

1. 배우기 쉬운 객체지향 언어
2. 자동 메모리 관리 (가비지 컬렉터 - GC)
3. 멀티쓰레드 지원
   - 하나의 프로그램에서 동시에 여러 작업을 할 수 있게 해준다.
   - ex) 채팅하는 친구와 채팅도하고 자료를 주고 받을 수 있다.
4. 풍부한 라이브러리로 쉽게 개발가능
5. 운영체제에 독립적
   - 자바 가상머신(JVM) 때문이다.
   - 자바 프로그램이 실행되는 가상 컴퓨터(VM)
